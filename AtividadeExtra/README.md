Atividade Extra - Controle de Vendas DiáriasEste projeto é uma aplicação desktop desenvolvida em C# utilizando o framework Windows Forms. O objetivo principal da atividade é praticar a manipulação de estruturas de dados homogêneas, especificamente o uso de arrays unidimensionais (vetores) e arrays bidimensionais (matrizes).A aplicação coleta o valor de vendas de 5 produtos diferentes para cada um dos 7 dias da semana, calcula os totais por dia, o total geral e exibe os resultados organizados em uma lista.🚀 FuncionalidadesEntrada Dinâmica de Dados: Utiliza caixas de diálogo (InputBox da biblioteca Microsoft.VisualBasic) para interagir com o usuário e receber os valores.Armazenamento em Matriz: Organiza os dados coletados em uma matriz bidimensional de 7 linhas (dias da semana) por 5 colunas (produtos).Validação de Dados: * Impede o fechamento inesperado do programa (crash) caso o usuário digite letras ou caracteres inválidos (double.TryParse).Bloqueia a inserção de valores negativos.Permite a reentrada imediata do dado caso ocorra um erro de digitação, sem perder o progresso do dia atual.Cálculo Automatizado: Acumula os valores para gerar o total de vendas por dia e o faturamento total geral.Visualização dos Resultados: Exibe os dados consolidados formatados monetariamente em um componente ListBox.Limpeza de Tela: Opção de limpar o histórico de exibição para reiniciar o processo.🛠️ Estrutura de Dados UtilizadaNo código principal (Form1.cs), foram aplicadas as seguintes estruturas:string[] diasSemana: Vetor utilizado para mapear e exibir textualmente os dias (de Domingo a Sábado).double[,] valores: Matriz $7 \times 5$ responsável por armazenar individualmente o preço/venda de cada produto por dia da semana.double[] totalDias: Vetor que armazena a soma das vendas de cada um dos 7 dias de forma indexada.📋 Como Funciona o AlgoritmoO usuário clica no botão Executar.O programa inicia dois laços de repetição aninhados (nested loops):O laço externo percorre os 7 dias da semana.O laço interno percorre os 5 produtos.A cada iteração, um InputBox solicita o valor do produto corrente.Após as validações de consistência do número, a matriz e os totalizadores são alimentados.Ao finalizar os produtos de um dia, o total acumulado daquele dia é adicionado ao ListBox.Ao final de toda a coleta, a linha de "Total Geral" é exibida.💻 Pré-requisitos e Como ExecutarPara rodar e estudar este projeto, você precisará de:Visual Studio (Recomendado versão 2022 ou superior).Carga de trabalho de Desenvolvimento de desktop com o .NET instalada.Certificar-se de que a referência para o namespace Microsoft.VisualBasic está ativa no projeto para permitir o uso do InputBox.Passos:Clone ou baixe os arquivos do projeto.Abra o arquivo de solução (.sln) no Visual Studio.Pressione F5 ou clique em Start/Iniciar para compilar e rodar a aplicação.Sinta-se à vontade para ajustar o título ou adicionar o nome da sua instituição de ensino caso ache necessário
